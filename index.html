Języki programowania

Język programowania to język rozumiany przez programistów.

Proces tłumaczenia kodu napisanego w danym języku na kod rozumiany przez komputer nazywa się kompilacją.

Języki programowania dzielimy na niskopoziomowe i wysokopoziomowe.

Niskopoziomowe

  • Operowanie bezpośrednio na pamięci komputera
    •
  • Bliskie powiązanie ze sprzętem
    •
  • Wymagana znajomość architektury sprzętowej
    •
  • Dużo trudniejsze niz języki wysokopoziomowe
    •

Przykłady: assembler, C

Wysokopoziomowe

  • Daje dużo większe wsparcie programiście w codziennej pracy
    •
  • Nakłada warstwy abstrakcji na sprzęt - nie wymaga znajomości architektury
    •
  • Daje łatwiejsze zarządzanie pamiecią
    •
  • Łatwiejszy w użyciu niż języki niskopoziomowe
    •

Przykłady: Java, Go, JavaScript, C++

Podstawy składni

Klasa

definiuje typ

składa się z pól i metod


public class Person {

    private int age;

    public int calculateYearOfBirth() {
        return 2019 - age;
    }

}

camelCase - konwencja nazewnicza Javy


int nameOfVariable;
String productId;
Long countOfProduct;

Indentyfikatory

nazwy zmiennych, pól, metod itd.

  • Dopuszczalne są litery, cyfry. Znak "_" traktowany jest jako litera
  • Wielkość liter ma znaczenie
  • Dopuszczalne są znaki narodowe (ale ich używanie jest złą praktyką!)
  • Brak ograniczenia długości
  • Identyfikatory nie mogą być słowami kluczowymi Javy

Wszystkie identyfikatory piszemy w języku angielskim!
